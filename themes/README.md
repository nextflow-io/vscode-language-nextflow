# Seqera themes

Bundled color and product icon themes for the Nextflow VS Code extension.

## Preview locally

1. Open this repository in VS Code or Cursor.
2. Press `F5` to launch the Extension Development Host.
3. **Preferences: Color Theme** → **Seqera Dark** or **Seqera Light**
4. **Preferences: Product Icon Theme** → **Seqera** (optional)

Open [`demo.html`](demo.html) in a browser for a side-by-side color theme preview.

Webviews and the DAG preview inherit `--vscode-*` CSS variables from the active theme automatically.
